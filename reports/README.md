# VOC 리포트 프롬프트 템플릿

매달 Claude에게 아래 내용을 복붙해서 사용하세요.

---

아래 엑셀 데이터를 바탕으로 월별 VOC 리포트를 작성해줘.

## 데이터 조건
- 문의량: 알브 상담 제외한 수치
- VIP 인입: VIP 전화 + 채팅 포함
- 조회 대비 문의량: 문의량 / 환급금 조회수
- 환급금 조회수: [직접 입력] ← 매달 여기에 숫자 입력
- CSAT: profile.csat 값 평균
- CSAT 참여율: profile.csat 건수 / 문의량
- 만족도 참여수: profile.csat 건수

## 분석 항목
1. 주요 데이터 요약 (문의량, VIP 인입, CSAT 등)
2. 전월 대비 변동 분석
3. 단계별 인입량 흐름 분석
4. 이번 달 주요 이슈 및 특이사항
5. 다음 달 개선 제안

## 리포트 형식
- 핵심 수치는 표로 정리
- 전월 대비 증감은 % 와 화살표(↑↓)로 표시
- 마지막에 한 줄 요약 작성
