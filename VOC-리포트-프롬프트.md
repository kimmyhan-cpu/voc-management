## ✉️ 프롬프트 본문 (여기서부터 복붙)

```
첨부한 채널톡 엑셀 데이터를 분석해서 VOC 리포트 HTML 파일을 만들어줘.

## 분석 기간
- 기간: [YYYY-MM-01] ~ [YYYY-MM-DD]

## 전월 수치 (추이 표 비교용)
- 문의량: [N]건
- VIP 인입: [N]건
- CSAT: [N]점
- CSAT 참여율: [N]%
- 환급금 조회수: [N]건 (별도 확인 후 입력)

## 데이터 조건 (UserChat 시트 기준)
- 문의량: UserChat 시트 전체 행 수 (필터링 없음)
  ※ 이 파일은 상담사가 처리한 전체 문의 내역이며 ALF 거쳐 상담사로 연결된 건도 포함
- VIP 인입: tags 컬럼에 VIP상담 포함 건수
- CSAT: profile.csat 컬럼 평균값
- CSAT 참여율: profile.csat 건수 / 전체 문의량
- 환급금 조회수: [N]건 (VOC 데이터에 없으므로 별도 확인 필요)

## 태그 → 단계 분류 기준
환급완료: 경정청구/결제 계열, 경정청구/환급금수령 계열, 경정청구/국세충당, 경정청구/농특세, 경정청구/지방세, 경정청구/원복, 단계별/결과신청완료, 단계별/신청완료 등
신청전: 경정청구/조회및신청, 경정청구/폐업, 경정청구/환급금 계열, 경정청구/조회_ 계열, 경정청구/계좌, 경정청구/마케팅 계열, 경정청구/충당, 경정청구/신청불가, 경정청구/수수료(신청전) 등
신청후~신고전: 경정청구/취소, 경정청구/환급불가, 경정청구/진행일정 계열, 경정청구/지수 계열, 경정청구/재진행, 경정청구/계좌/수정 등
신고후~환급결정전: 경정청구/환급일정 계열, 경정청구/환급금/변동, 경정청구/환급금/상세내역, 경정청구/가산금, 경정청구/세무서요청, 경정청구/환급불가/기각, 단계별/결과신고완료 등
결제완료: 경정청구/신고서, 경정청구/현금영수증, 경정청구/세금계산서, 경정청구/수정신고, 경정청구/환불
계정: 회원&계정 계열, 개인정보보안
서비스문의: 기타/타서비스, 이벤트, 비즈넵환급, 기타/세무상담, 기타/세나, 비사업자 등
기타: 기타/문의미확인, 케어문의, 기타/테스트, 영업/제휴 등

## 리포트 구성
1. KPI 카드 4개: 문의량(전월대비↑↓표시), VIP 인입, CSAT, CSAT 참여율
2. 신청 단계별 인입량 카드
   - 경정청구 단계: 환급완료 / 신청전 / 신청후~신고전 / 신고후~결정전 / 결제완료
   - 기타 분류: 계정 / 기타 / 서비스문의 / 미분류
   - 카드 클릭 시 세부 태그 TOP5 펼쳐지는 인터랙션
3. 단계별 문의 비중 가로 바 차트
4. 월별 주요 지표 추이 표 (전월 포함, 셀 직접 수정 가능, 행 추가 가능)
5. 이달의 이슈 (유형/제목/날짜/상태/내용 입력 및 추가/삭제 가능)
6. 이번달 개선 내역 (우선순위/항목명/결과 입력 및 추가/삭제 가능)
7. 이달의 총평 (textarea 직접 입력 가능)

## 기능 요구사항
- 헤더 제목과 기간은 클릭해서 직접 수정 가능 (contenteditable)
- 🔗 링크 복사 버튼
- PDF 저장 버튼: 클릭 시 모든 단계 세부 내용 펼쳐진 상태로 저장
- PDF 저장 시 버튼/입력폼/삭제버튼 숨김 처리 후 캡처, 캡처 후 원상복구
- 브라우저에서 열 때는 모든 입력/편집 기능 정상 작동
```
